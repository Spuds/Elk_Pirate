[color=blue][b][size=12pt][u]Introduction[/u][/size][/b][/color]
This adds a very basic English text to pirate lexer and will convert your sites posts in to pirate speak which is bountifully bombastic me hearties!Â  The conversion is only for the display of the post, it does not change the actual post, so once the addon is disabled things will return to normal.
